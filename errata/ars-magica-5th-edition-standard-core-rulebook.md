# Ars Magica 5th Edition STANDARD Core Rulebook Errata

Canonical errata source for the core rulebook pilot.

## Source Status

- Official source: Atlas Games, Ars Magica Fifth Edition Errata
- URL: <https://www.atlas-games.com/arsmagica/arm5errata>
- Local PDF matched: `Ars_Magica_-_5th_Edition_Core_Rulebook_[AG0205].pdf`
- Working assumption: this PDF is likely an early or first-printing file because its PDF metadata is dated July 6, 2005

## Canonical Errata To Apply

### First-Printing Baseline

- Credits: add Andrew Mitchell to Fifth Edition Design Contributions.
- Die Rolls: correct `role` to `roll`.
- Intelligence: replace the first sentence and expand the last sentence to clarify what Intelligence does and does not represent.
- Specialist and Knight templates: correct several Ability and Initiative values.
- Combat and tables: fix multiple statistics and formula issues in the combat, books, and advancement sections.
- Character sheet: use the corrected sheet linked from the official errata page.

### Later Core Updates Worth Tracking

- Priest, Rogue, Bjornaer, Bonisagus, Jerbiton, Tremere, and Verditius templates all receive substantive corrections in the newer second-printing errata stream.
- Fast Casting, Penetration, Certamen, and Verditius Magic all receive rules-text revisions that are important for play reference.
- Multiple Virtues and Flaws receive wording corrections that affect character creation and laboratory totals.
